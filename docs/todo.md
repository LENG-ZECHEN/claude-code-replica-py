## Overview

The TodoWrite V1 surface replicates Claude Code's single-tool, in-memory
session todo list and its `<system-reminder>`-driven stale-todo nudge.
The model expresses task intent declaratively through one tool
(`todo_write`); the engine teaches it when to call via a static
`## Todo Management` system-prompt section, observes drift via a
double-AND turn counter, and prepends a per-turn
`ATTACHMENT_TODO_NUDGE` USER message when both
`turns_since_write >= N` and `turns_since_reminder >= N` hold. The
6-tool V2 Tasks suite (file persistence under `.tasks/`,
`.highwatermark` lockfile concurrency, swarm-shared task list,
blocks/blockedBy DAG, owner field) is explicitly out of scope — in TS
the two surfaces are mutually exclusive via the `isTodoV2Enabled()`
gate, and V1's single-process / in-memory shape matches the replica's
immutable-transcript style.

## Public API

- Tools: `todo_write` (registered in `_build_repl_loop` only when
  `todo_nudge_enabled=True`).
- Messages: `MessageType.ATTACHMENT_TODO_NUDGE` +
  `Message.attachment_todo_nudge(content)`.
- AgentLoop kwargs: `todo_nudge_enabled: bool = True`,
  `todo_reminder_turns: int = 10`, `todo_state: list[TodoItem] | None = None`.
- ContextBuilder: `build(todo_nudge: TodoNudge | None = None)`.
- Metrics: `MetricsCollector.todo_writes` /
  `MetricsCollector.todo_nudges_armed` (with
  `record_todo_write()` / `record_todo_nudge_armed()`).
- Trace: `tracer.emit("todo", ...)` channel.
- CLI: `--no-todo-reminder` / `--todo-reminder-turns N` on both
  `simple-agent` and `simple-agent-openai` REPLs.
- REPL: `/todos` slash command (glyph-prefixed list).

## Boundaries

- Reads from: `Transcript.all_messages()` (for the parity-test
  reverse-scan helpers) and `AgentLoop._todos`.
- Writes to: `AgentLoop._todos` via the `set_todos` closure;
  `MessageType.ATTACHMENT_TODO_NUDGE` injected by
  `ContextBuilder.build()`; `MetricsCollector.todo_writes` /
  `todo_nudges_armed`.
- Depends on: `models.py` (Message, MessageType, Role, ToolCall),
  `tools.py` (Tool, ToolRegistry).
- Does NOT do: file persistence (`shouldDefer=true` lazy-load,
  `.tasks/` JSON files), DAG blocks/blockedBy, per-(agentId | sessionId)
  namespacing, V2 6-tool suite, BRIEF mutex short-circuit,
  `CLAUDE_CODE_DISABLE_ATTACHMENTS` global switch, or
  `verificationNudgeNeeded` branching.
